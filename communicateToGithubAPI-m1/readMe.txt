The project is based on 
-android platform, 
-kotlin language,
-git bash and github for versionning,
-gitbug API
-somes android librairies: Moshi, Room, Glide
- and others.
The project connect to the REST-github API, specially the users json data interface.
Get those data with Moshi API as json and adapt them as user local object.
Search a particular user by his login and display some of his information like his
avatar with Glide librairy. Save them in local database, using singleton user pattern with 
Room librairy. Kotlin and all his functonnalities have been used frequently.
